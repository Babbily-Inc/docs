The Library helps you find work you have already created in Babbily. It can include saved chats, translations, images, videos, and other media.

## Find saved work

Open **Library** to browse and search saved items. You can filter by content type and use item actions to reopen, rename, share, download, or delete supported items.

Library filters can include:

- Threads.
- Images.
- Audio.
- Videos.
- Translations.

## Share a chat

When sharing is available for a chat, use the share action on the thread. Babbily creates a public share view that others can open with the share link.

Temporary Chats are not shareable. Use a regular chat if you need to create a share link.

Shared chats are useful for:

- Sending a research summary to a teammate.
- Sharing a generated answer without inviting someone into your account.
- Showing a public version of a conversation.

## What shared viewers can see

People with the link can see the messages plus any attachments, generated media, citations, or other content included in the shared thread. They do not get access to your account, private settings, connectors, billing, or memory controls.

<Warning>
  Review a thread before sharing it. Do not share chats that include confidential, personal, or sensitive information.
</Warning>

## Public search visibility

The share dialog lets you make a link public or private. When a thread is public, the dialog notes that it can be found in Google Search Results. When you make a link private, Babbily removes the thread from public search visibility where that control applies.

<Note>
  Search engines can take time to update. If a thread included sensitive information, make it private and contact support if you need help.
</Note>

## Delete saved items

Deleting an item removes it from your Library. Some deletion actions cannot be undone.

<Tip>
  If you want a chat that does not appear in Library, use [Temporary Chat](/features/temporary-chat) before you begin. If you want to find saved translation work, open [Translate](/features/translate) or **Library > Translations**.
</Tip>
