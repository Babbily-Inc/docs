Temporary Chat is for conversations you do not want saved in normal chat history or Library.

## When to use Temporary Chat

Use Temporary Chat for:

- One-off brainstorming.
- Drafts you do not need later.
- Quick questions.
- Sensitive context you do not want saved in your normal history.

<Warning>
  Temporary Chat helps keep a conversation out of saved chat surfaces, but you should still avoid entering passwords, keys, payment details, or other secrets.
</Warning>

## Start Temporary Chat

<Steps>
  <Step title="Open the Temporary Chat control">
    Use the Temporary Chat control in the chat interface.
  </Step>
  <Step title="Confirm Temporary Chat">
    Start the temporary session.
  </Step>
  <Step title="Chat normally">
    You can still use supported models, tools, attachments, image generation, and video generation.
  </Step>
  <Step title="Exit when finished">
    Starting a new chat or leaving the temporary thread exits Temporary Chat and removes it from saved chat surfaces.
  </Step>
</Steps>

## What changes in Temporary Chat

- The thread does not appear in Recents or Library.
- The thread is not meant for sharing.
- Active generation may be stopped when you exit.
- The session can restore in the same browser tab while it is active.

<Tip>
  If you realize you want to keep the work, copy the useful result into a normal chat or document before exiting Temporary Chat.
</Tip>
