## What you can usually save or export

| Content | Customer action |
| --- | --- |
| Chat text | Copy important messages before deleting or leaving a temporary workflow. |
| Translations | Copy the translated text or continue the translation in chat when available. |
| Generated images | Save or download from the image controls when available. |
| Generated videos | Save, preview, fullscreen, or download when controls are available. |
| Shared work | Copy the share link when sharing is enabled. Review the shared thread before sending it. |
| Source citations | Open citations and save source links separately when needed. |

<Note>
  Available actions can vary by content type, model, plan, browser, and account state.
</Note>

## What you can usually delete or disconnect

- Delete supported saved items from **Library**.
- Delete or update memory from **Settings > Memory** when controls are available.
- Ask Babbily to forget eligible saved context in chat.
- Disconnect apps from **Settings > Connectors** when controls are available.
- Revoke connected-app access from the connected app's own settings.
- Delete shared links or saved work where the product exposes a delete action.

## Account deletion and support requests

For account deletion or data you cannot remove yourself, contact support. Include the account email, the type of data involved, what you already tried, and whether the request is for an individual account or a team workspace.

Some information may need to be retained where required for billing, security, legal, support, abuse-prevention, or operational reasons.

## Before deleting

<Steps>
  <Step title="Save what you still need">
    Copy or download important output before deleting a chat, file, media item, or Library entry.
  </Step>
  <Step title="Review shared access">
    Check whether a chat or item has been shared with others.
  </Step>
  <Step title="Review memory">
    Open **Settings > Memory** and remove context you no longer want Babbily to reuse.
  </Step>
  <Step title="Review connectors">
    Disconnect or revoke connected-app access you no longer want to use.
  </Step>
  <Step title="Contact support if needed">
    Ask support for help with account-level deletion or anything not available in the UI.
  </Step>
</Steps>

<Warning>
  Deletion can be permanent. Save copies of work you need before removing items.
</Warning>
