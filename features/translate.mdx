Translate gives you a focused place to translate text without setting up a full chat first.

## Translate text

<Steps>
  <Step title="Open Translate">
    Select **Translate** from the sidebar.
  </Step>
  <Step title="Choose languages">
    Choose the source language and target language. You can also use auto-detect when it is available.
  </Step>
  <Step title="Enter text">
    Type or paste the text you want translated.
  </Step>
  <Step title="Review the translation">
    Babbily translates after you stop typing. You can copy the result or continue in chat.
  </Step>
</Steps>

## Choose a translation style

Use style chips when you want the translation to sound a certain way:

- More fluent.
- Business formal.
- Simple enough to explain to a child.
- Academic.

Select the same style again to remove it.

## Use voice input

Use the microphone button in the source text panel to dictate text. Voice input depends on browser, device, microphone permission, and account state.

If voice input is unavailable, type or paste your text instead.

## Continue in chat

After a translation appears, use **Continue in chat** to open a new chat with the original text and translated text included as context.

This is useful when you want to:

- Ask why a phrase was translated a certain way.
- Rewrite the translation for a different audience.
- Turn the translation into an email, post, or document.

## Translation history

Signed-in customers can find saved translations in **Library > Translations**.

<Tip>
  Translation uses your API usage budget. For long documents, translate one section at a time so you can review tone and formatting as you go.
</Tip>
