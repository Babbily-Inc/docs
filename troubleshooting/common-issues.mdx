## A tool is missing

Try these steps:

<Steps>
  <Step title="Check the model">
    Some tools appear only for models that support them. Switch models if the tool is hidden.
  </Step>
  <Step title="Check tool mode">
    Use **Auto** or **Manual**. In **Off** mode, Babbily does not call tools.
  </Step>
  <Step title="Check your plan">
    Some capabilities can depend on your plan or remaining usage.
  </Step>
</Steps>

## A connector is not ready

If a connector shows setup, authorization, or preparation status:

- Finish the connector setup flow.
- Refresh **Settings > Connectors**.
- Reconnect if authorization changed.
- If you do not see a disconnect option, revoke access from the connected app's own settings and contact support if you still need help.
- Wait a moment if the connector was just connected.

## Babbily did not search

If you expected live information, switch to **Manual** and choose **Web search** or **Deep research**. You can also ask directly:

```text
Search the web and cite sources for the latest information about this topic.
```

## Memory feels wrong

Open **Settings > Memory** and review the Memory Profile and citations. Remove anything outdated. You can also say:

```text
Forget that I prefer long answers.
```

If a memory cannot be removed from the UI, contact support.

## Usage looks delayed

The usage card may take a short time to refresh after model, tool, image, or video work. Wait a moment and refresh the page. If usage still looks wrong, check **Settings > Billing**.

## A chat is near the context limit

Start a new thread if the topic changed or the context indicator is full. Babbily may compact older context, but very long threads can still become harder to steer.

## Image or video generation failed

Try a shorter, clearer prompt. If you attached source media, confirm that the selected model supports it. Also check your usage budget before trying again.

## Translate is not updating

Try these steps:

- Make sure a translation model is available in the model picker.
- Check that the source text is not empty.
- Wait a moment after you stop typing.
- Try a shorter section of text.
- Check your API usage budget if translation stops with a usage message.

## Voice dictation is unavailable

Voice dictation depends on your browser, device, microphone permission, and account state.

If the microphone button does not work, allow microphone access, close other apps that may be using the microphone, refresh Babbily, or type the text manually.

## News is empty or not local

If News does not show what you expect:

- Try another category.
- Set or update your location in **Settings > Profile**.
- Refresh News after updating your location.
- If no profile location is set, News may use an approximate or default location.
- Open the original sources before relying on a story for important decisions.

## Billing controls are missing

Billing controls can depend on your role. Team members may not see owner-only billing actions. Ask the team owner to manage subscription changes.

## Login or invite access fails

Use the latest verification code or invitation email, confirm the account email, and finish onboarding if prompted. If the team still does not appear after accepting an invite, ask the team owner to check the invited email and pending invitations.
