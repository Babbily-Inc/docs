## Quick checks

| Issue | Try this first |
| --- | --- |
| Code did not arrive | Check spam or promotions folders, then request a new code. |
| Code is not accepted | Use the newest code and enter it exactly as shown. |
| Account not found | Confirm the email address and sign-in method you used before. |
| Account already exists | Switch from **Sign Up** to **Log In**. |
| Invite link does not work | Ask the team owner to confirm the invited email and send a fresh invite. |
| Share link sends you to login | Sign in, then open the share link again. |

## Verification code problems

Verification and password reset codes are time-sensitive. If a code fails:

- Use the most recent email from Babbily.
- Check that you are entering the code for the same email address shown on the page.
- Request a new code if the original code is old.
- Avoid copying extra spaces before or after the code.

If codes are not arriving, check spam filters and ask your email administrator whether automated messages are being filtered.

## Password reset problems

If the reset page says a reset code is required, restart from **Forgot your password?** on the sign-in page. This helps Babbily verify which email address should receive the reset code.

If your new password is not accepted, follow the password requirements shown on the page and make sure the confirmation field matches.

<Note>
  If you normally sign in with a social sign-in option, use that option again. A password reset may not apply unless you added email and password sign-in to the account.
</Note>

## Team invitation problems

If a team invitation does not work:

<Steps>
  <Step title="Use the latest invitation">
    Older invitation emails may no longer be valid after a team owner sends a new one.
  </Step>
  <Step title="Check the invited email">
    The invitation is intended for the email address the team owner entered.
  </Step>
  <Step title="Complete onboarding">
    New users may need to finish onboarding before Babbily can add them to the team.
  </Step>
  <Step title="Ask the owner to resend">
    If the link is expired, missing, or for the wrong email, ask the team owner to send a new invite.
  </Step>
</Steps>

## You can sign in but do not see the team

Try these checks:

- Open **Settings > Team** and refresh the page.
- Confirm you accepted the invite with the same email the owner invited.
- Ask the team owner whether the invite is still pending, accepted, cancelled, or sent to another email.
- Ask the owner to remove duplicate or outdated invitations if needed.

## When to contact support

Contact support if:

- You cannot receive any verification or reset messages.
- Your invite was accepted but team access is still missing.
- You are locked out of the account email.
- You need help with account deletion or data you cannot remove yourself.

<Warning>
  Do not send passwords, verification codes, private keys, payment details, or raw connected-app credentials to support.
</Warning>
