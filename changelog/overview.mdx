<Update label="June 7, 2026">

## New features

**Skills, tools, memory, auto mode, and connectors.** Babbily can now personalize replies with persistent [memory](/features/memory), use [tools and skills](/features/tools-and-skills) automatically when they help, and pull live context from [connected apps](/features/connectors). Auto mode picks the right capability for the task without manual toggling.

**API usage billing.** A new monthly [API usage budget](/features/usage-budget) replaces fixed message limits. Track what you've used, see what counts, and know when it resets — all from the chat sidebar.

## Updates

**Clearer context indicator.** The [context window](/features/context-window) tracker now updates more reliably during long chats so you can tell when Babbily will compact older turns.

**Budget refresh.** Your remaining budget refreshes faster after each turn, so the number you see matches what you've actually used.

## Fixes

- Resolved a stall when the token-limit warning appeared mid-send.
- Restored model fallback so [chat](/features/chat-and-models) keeps working when a preferred model is briefly unavailable.
- Fixed scrolling and layout glitches in [temporary chats](/features/temporary-chat) and on tablet viewports.
- Sign-in via Google and Apple is now more reliable on mobile.

</Update>
